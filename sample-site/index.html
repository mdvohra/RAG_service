This page embeds the RAG chat widget in the bottom-right corner.

Welcome

This is a minimal sample page for testing your RAG Embed widget locally. Upload documents in the setup wizard, then ask questions about them using the chat bubble.

Try asking

  • What is your refund policy?
  • How can I contact support?
  • What services do you offer?
Open the purple chat button in the bottom-right. If the widget does not appear, ensure the API is running at http://localhost:8000 and the widget bundle exists (widget/dist/embed.js).

How to open this page

From the project root:

cd sample-site
npx --yes serve -l 3000

Then visit http://localhost:3000 (any localhost port works with default settings).
